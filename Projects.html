Chess AI Solo Project

This is a solo project I completed for my DAM101 final assignment in the second semester. I developed a Chess-playing AI using the Negamax search algorithm with Alpha-Beta pruning as its foundation. To improve performance and evaluation accuracy, I implemented Transposition Tables to cache board evaluations and added Quiescence Search to handle volatile positions more effectively.

Snapchat Clone Group Project

This is a group project where we built a Snapchat clone using React and Next.js. We developed a fully functional frontend and integrated it with a complete backend. Key features include user authentication, a camera interface, sending and accepting friend requests, and real-time messaging.
